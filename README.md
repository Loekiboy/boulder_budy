# Boulder Buddy

A site with a pet rock simulation where users can design, feed, and pet their own rock.

**Play the project here:**
https://loek.oerlemans.tv/boulder_budy/

---

## Features

* **Custom Design:** Users can drag and drop eyes and a mouth onto the rock to create their own unique buddy.
* **Interaction:** Includes dedicated actions for feeding and petting the rock.
* **Communication:** The rock displays text bubbles to express its needs, such as hunger or a request for attention.
* **Hunger Mechanics:** A hunger bar tracks the status of the rock. If the hunger drops, the rock will prompt the user to take action.

---

## Technical Overview

The project uses standard HTML5, CSS3, and vanilla JavaScript to handle user inputs and interactions.

---

## How to run locally

1. Clone this repository to your local machine.
2. Open `index.html` in any modern web browser.
